# sesiones/

Cada sesión vive en su carpeta. La convención es la misma para las cuatro:

```
sesion-N-nombre-corto/
├── README.md              ← índice y estado de la sesión
├── objetivos.md           ← objetivos aprendizaje + criterios de éxito
├── guion.md               ← guion del facilitador con tiempos
├── previo.md              ← fuente markdown del PDF previo
├── slides/
│   └── index.html         ← presentación reveal.js (o equivalente)
├── ejercicios/
│   ├── ejercicio-N.md     ← instrucciones del ejercicio
│   └── ...
├── herramientas/
│   └── [nombre].html      ← herramienta interactiva autocontenida
├── material-participante.md  ← guía de referencia post-sesión
└── evaluacion.md          ← preguntas específicas de la sesión
```

No todas las subcarpetas se crean de entrada. Se van creando conforme la sesión se desarrolla.
